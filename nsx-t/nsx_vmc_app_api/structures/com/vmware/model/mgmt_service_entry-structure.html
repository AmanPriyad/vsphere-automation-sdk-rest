    model: mgmt service entry

    A service entry describes the detail of a network service.

    Representation:

    • JSON
    {
        "path" : "string",
        "display_name" : "string"
    }

    Attributes:

    Name Type Description
    Optional
    path string Service path should refer to a valid service in the system. Service can be system defined or user defined.

    Optional.

    display_name string Display name for this service
