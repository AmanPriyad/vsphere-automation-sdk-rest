    api default: list connected services

    List services connected to this linked vpc, for example, S3. The response consist of all available services along with their status.

    Request:

    HTTP request

    GET https://{server}/rest/cloud-service/api/v1/infra/linked-vpcs/{linked_vpc_id}/connected-services

    Path Parameters

    Name Type Description
    Required
